Incrementing & Decrementing

Offline lesson

Update changing values

Programs often need to increase or decrease a value. This is called incrementing and decrementing. Python's += and -= operators make those updates shorter and easier to read.

01 Start by experimentingChallenge

Run the code in Runnable 1 and observe what happens. Then explore how += and -= affect the variable.

  1. Predict the output before running the script, then run it to check.
  2. Change counter += 3 to use a different amount. What output do you expect?
  3. What does counter -= 2 do, and how does it change the final output?
02 Understand what happenedExplanation

The counter starts at 5.

counter += 3 changes it to 8.

counter -= 2 then changes it to 6.

Incrementing and decrementing operators

The operators are shorthand for reading a variable's current value, performing an operation, and assigning the new value back to that variable.

ExpressionEquivalent expression
score += 2score = score + 2
balance -= 5balance = balance - 5

Where these updates are useful

Tracking a score

score = 0
score += 10  # Player earns 10 points
print(score)  # 10

Managing a budget

balance = 100
balance -= 25  # User spends 25
print(balance)  # 75

Adjusting product stock

stock = 50
stock -= 3  # Three items sold
print(stock)  # 47
03 Build something yourselfTest your knowledge

In Runnable 2, build a fitness-app step tracker. Write a script that:

  1. Starts with 0 steps.
  2. Adds 3000 steps.
  3. Adds 2000 more steps.
  4. Removes 500 steps to correct an error.
  5. Prints the final total.
Starting value0 steps
Expected outputTotal steps: 4500
04 Compare after attempting itSolution

Your code may differ, but it should follow this structure:

# Write your code here:
steps = 0  # Start with zero steps

# User logs steps
steps += 3000
steps += 2000

# User removes an incorrect entry
steps -= 500

print("Total steps:", steps)
05 Take the idea forwardReview

Using += and -= makes changing values concise and readable. These operators are common in scores, budgets, inventory systems, shopping carts, fitness apps and loops.
