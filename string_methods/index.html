String Methods

Prototype lesson

Work with text efficiently

Strings are one of the most commonly used data types in Python. Python provides built-in string methods that modify, format and analyse text.

02 Understand what happenedExplanation

.upper() returns "HELLO, PYTHON!".

.lower() returns "hello, python!".

.replace("Python", "World") returns "Hello, World!".

The replacement is case-sensitive. Using "python" will not match "Python", so the text remains unchanged.

The original string does not change

String methods return a new, modified string. To keep the result, assign it to a variable:

text = text.upper()  # overwrite text with the returned value

What is a method?

A method is a prewritten function attached to a particular type of data. It provides a ready-made way to perform a common task. For example, the string method .lower() changes every letter in a string to lowercase.

Saving string method values

Store a method's returned value when you want to use it later:

email = "User@Example.COM"
lower_case_email = email.lower()
print(lower_case_email)  # "user@example.com"

Common string methods

.lower()

Changes all letters to lowercase.

email = "User@Example.COM"
print(email.lower())
"user@example.com"
.upper()

Changes all letters to uppercase.

username = "pythonDev"
print(username.upper())
"PYTHONDEV"
.replace()

Replaces exact matching text with something else.

sentence = "I love JavaScript!"
new_sentence = sentence.replace("JavaScript", "Python")
print(new_sentence)
"I love Python!"
.strip()

Removes whitespace from the beginning and end.

user_input = "  Hello, World!  "
cleaned_text = user_input.strip()
print(cleaned_text)
"Hello, World!"

Python contains many more string methods. See the official Python documentation when you are online.

04 Compare after attempting itSolution

Your code might differ, but it may follow this structure:

name = input("Enter your name: ")

cleaned_name = name.strip()
lower_name = cleaned_name.lower()
email = lower_name.replace(" ", "") + "@email.com"

print("Email address:", email)

Method chaining

Once each step makes sense, the methods can be joined in one expression:

email = input("Enter your name: ").strip().lower().replace(" ", "") + "@email.com"
print("Email address:", email)

Shorter is not automatically clearer. Writing the stages separately is useful while learning what each method returns.

05 Take the idea forwardReview

String methods help clean user input, format text for display and manipulate data consistently. They are useful for form submissions, email addresses, clean URLs and many other everyday programming tasks.
