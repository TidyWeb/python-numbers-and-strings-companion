Basic Math with Arithmetic Operators

Offline lesson

Calculate with Python

Math is everywhere in everyday life, from calculating the total cost of groceries to determining how long a trip will take. Python arithmetic operators perform those same calculations in code.

02 Understand what happenedExplanation

6 + 4 returns 10.

10 - 3 returns 7.

5 * 2 returns 10.

8 / 4 returns 2.0.

(2 + 3) * 2 returns 10.

Parentheses change the order of a calculation: everything inside them is evaluated first. Division with / always returns a floating-point number, so 8 / 4 produces 2.0.

Understanding arithmetic operators

OperatorDescriptionExampleResult
+Addition3 + 25
-Subtraction7 - 43
*Multiplication6 * 318
/Division; returns a float8 / 24.0
//Floor division; rounds the whole-number result down9 // 24
%Modulus; returns the remainder10 % 31
**Exponentiation; raises a value to a power2 ** 38

Order of operations (PEMDAS)

  1. Parentheses ()
  2. Exponents **
  3. Multiplication, division, floor division and modulus * / // %, evaluated left to right
  4. Addition and subtraction + -, evaluated left to right
print(5 + 2 * 3)    # 11
print((5 + 2) * 3)  # 21
