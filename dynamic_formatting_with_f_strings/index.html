Dynamic String Formatting with F-Strings

Offline lesson

Put values straight into text

F-strings are a clear, modern way to insert variables, expressions and formatted numbers directly into Python strings.

02 Understand what happenedExplanation

The first line prints: Hello, my name is Morgan and I am 25 years old.

name holds "Morgan"; str(age) converts 25 into text.

An f-string begins with f and places values inside curly brackets: {}.

Why use f-strings?

Concatenation needs several + operators and manual conversions such as str(age). An f-string lets the sentence remain in one readable piece:

name = "Morgan"
age = 25
print(f"Hello, my name is {name} and I am {age} years old.")

This is useful for user information, reports, dashboards and messages whose values change while the surrounding text stays the same.

Common f-string uses

UseExampleOutput
Insert variablesf"The {product} costs ${price}."The Laptop costs $799.
Evaluate expressionsf"Total cost: ${subtotal + tax}"Total cost: $55
Format decimalsf"Price: ${price:.2f}"Price: $3.50

04 Compare after attempting itSolution

Your code may differ, but it can follow this structure:

item_name = "Coffee"
price = 3.50
tax_rate = 0.13
tax_percentage = 13

total_price = price + (price * tax_rate)

print(f"Item: {item_name}\nPrice: ${price:.2f}\nTax: {tax_percentage}%\nTotal Tax: ${price * tax_rate:.2f}\nTotal after Tax: ${total_price:.2f}")

Read :.2f in pieces: the colon begins the formatting instruction, .2 requests two decimal places, and f means a fixed-point number.

05 Take the idea forwardReview

F-strings make dynamic output easier to read and maintain. They insert variables without manual type conversion, evaluate expressions inside {}, and format values such as prices precisely.

06 Optional companion practiceF-String Playground

You have now completed the original Code Institute lesson. This optional playground goes beyond it, so you can experiment with other useful forms of f-string formatting.

Open the Playground tab in the Python workbench. Run the starter code first, then change one example at a time and run it again.

1. Put several values into one string

Each pair of curly brackets contains a variable or expression that Python evaluates:

learner = "Sam"
course = "Python"
week = 3
print(f"{learner} is studying {course} in week {week}.")

Try it: change all three values. Notice that the number does not need str().

2. Calculate inside the braces

The braces can contain an expression—not only a variable name:

price = 12.50
quantity = 4
print(f"Order total: £{price * quantity:.2f}")

Try it: change the price and quantity, then predict the result before running the code.

3. Format decimals, percentages and large numbers

InstructionMeaningExample output
:.2fFixed-point number with two decimal places£12.50
:.0%Percentage with no decimal places13%
:,Thousands separators1,250,000
 
completion = 0.875
visitors = 1250000
print(f"Completed: {completion:.0%}")
print(f"Visitors: {visitors:,}")

Percentage trap: Python multiplies the value by 100 for percentage display. Store thirteen per cent as 0.13, not 13.

4. Transform text inside an f-string

A method call is also an expression:

name = "ada lovelace"
print(f"Welcome, {name.title()}!")

Try it: replace .title() with .upper().

5. Align text into columns

A width gives the value room to occupy. The symbols <, > and ^ align left, right and centre:

item = "Notebook"
price = 4.5
print(f"|{item:<12}|£{price:>7.2f}|")

Try it: add two more products and print them with the same widths.

6. Print curly brackets themselves

Single braces have a special job in an f-string. Double them when the output needs literal braces:

topic = "variables"
print(f"Use {{curly brackets}} around {topic}.")

Output: Use {curly brackets} around variables.
