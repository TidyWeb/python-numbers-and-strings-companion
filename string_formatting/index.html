String Formatting

Offline lesson

Build clear text

String formatting helps programs create readable messages, combine saved text, choose quotes safely and use escape characters for special formatting.

01 Start by experimentingChallenge

Before running Runnable 1, predict what each print statement will display.

  1. What exact text and punctuation will the first print statement produce? What does + do?
  2. Why can double quotes safely surround "It's a beautiful day!"?
  3. What does \n do inside a string? Try adding another one and observe the output.
  4. The final example also contains \l. Run it and notice how Python treats that unknown escape sequence.
02 Understand what happenedExplanation

+ joins strings together; this is called concatenation.

Double quotes can contain the apostrophe in It's without ending the string.

\n creates a new line.

Concatenating strings with +

Concatenation builds messages, labels and prompts from smaller pieces of text.

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)  # John Doe

Every joined value must be a string. Convert numbers with str() first:

age = 25
print("I am " + str(age) + " years old.")

Single and double quotes

Python accepts either quote style. If the text contains one type of quote, use the other type around the string:

print("It's a sunny day!")
print('She said, "Python is fun!"')

03 Build something yourselfTest your knowledge

In Runnable 2, display a welcome message and the location of a user's files:

  1. Store the user's name in a variable.
  2. Create a greeting that includes the name.
  3. Store a file path containing literal backslashes.
  4. Print the greeting and file path on separate lines.
Example userRihanna
Expected outputWelcome, Rihanna!
Your folder is located at C:\Users\Rihanna\Documents
04 Compare after attempting itSolution

Your code may differ, but it can follow this structure:

username = "Rihanna"
welcome_message = "Welcome, " + username + "!"
file_path = "Your folder is located at C:\\Users\\Rihanna\\Documents"

print(welcome_message)
print(file_path)

Remember: A single backslash begins an escape sequence. Use \\ when the output needs an actual backslash.

05 Take the idea forwardReview

Concatenation, quote choice and escape characters make text output flexible and predictable. They help prevent syntax errors in user messages, file paths and reports.
